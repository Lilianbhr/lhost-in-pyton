# lhost-in-pyton
Projet de fin d'année des terminale spécialité NSI 2026

## Structure du projet
Cette rubrique traite de la structure du projet, les branches comme les dossiers

### Branches
 - **branche principal** : main -> interdit d'y toucher, elle contient la dernière version fonctionnelle complète du projet
 - dev -> interdit d'y toucher, elle contient la dernière version fonctionnelle du projet, c'est depuis celle ci que vous récupérez les dernière modifications du projet global
 - tampon -> c'est sur celle ci que vous réalisez vos pull request pour ajouter votre fonctionnalité au code global, vous ne récupérez jamais de code sur cette branche
 - **fonctionnalitées**: vous pouvez créer autant de branche que vous voulez depuis *dev*, elles sont à vous, mais tout le monde peut y accéder, sauf si vous travaillez sur la branche en question, ne récupérez jamais de code dessus

### Dossiers
 - **racine du projet** : lhost-in-pyton -> il contient tout le projet et le *.git*
 - assets -> contient toutes les ressources de l'application, les images, les musiques, les polices etc... les ressources seront regroupées dans des sous dossiers
 - src -> contient tout le code du projet, *main.py* est le point d'entrée du programme
